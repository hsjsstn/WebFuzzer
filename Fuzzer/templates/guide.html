WebFuzzer 구성

  • 정적 크롤링: 사이트 링크 구조 분석 (HTML 파싱 기반)
  • 동적 크롤링: JavaScript 렌더링 후 동적 링크/폼 추출
  • 폼 수집: 탐지된 입력 필드/폼 구조 정리
  • 비동기 퍼징: 각 취약점 유형별 페이로드를 병렬 테스트
  • PDF 리포트 생성: 결과 요약 + 상세 내용 시각화
